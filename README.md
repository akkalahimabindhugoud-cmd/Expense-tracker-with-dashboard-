
# Expense Tracker with Dashboard

A simple Expense Tracker system that helps users manage income and expenses while displaying data through an interactive dashboard.

## Features

- Add income and expenses
- Edit and delete transactions
- Categorize expenses
- Display dashboard analytics
- Generate monthly reports

## Objectives

- Track daily expenses
- Manage spending effectively
- Visualize financial data
- Improve budgeting

## Workflow

User enters transaction details

↓  

Data stored in database

↓  

System processes expense data

↓  

Dashboard updates automatically

## Dashboard Modules

- Income Summary
- Expense Summary
- Pie Chart Analysis
- Monthly Expense Report
- Category Tracking
